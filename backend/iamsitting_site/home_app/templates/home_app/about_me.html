Senior Design

My project was called Cycle-X Pro. In a nutshell, it is a Fitbit + LifeAlert for your bicycle. Visit the embedded site to find out more; there is a good video about it as well. I worked on pretty much all the software including the C code for the Arduino, the Django website, and the Android app. Check out my GitHub to find out more.

Embedded Software Engineer

While I was in school, I worked as a part-time software engineer for a small company in Bryan, Texas called Otis Instruments, Inc. Our main products were gas detection units for sites with hazardous gases (i.e chemical plants, refineries, etc.). Because these units are digitial and wireless, there was need for someone like me. I was programming in C on PIC Microcontrollers to add any necessary features to pre-existing software. I really enjoyed what I was doing because I got to handle the hardware as well.
